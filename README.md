# MERN Docker Project

A full-stack application utilizing the MERN stack (MongoDB, Express, React, Node.js) with Docker for containerization. This setup aims to streamline development and deployment processes.

## 📦 Technologies Used

* **Frontend:** React (v18)
* **Backend:** Node.js (v18) with Express
* **Database:** MongoDB (latest)
* **Containerization:** Docker, Docker Compose
* **Web Server:** Nginx (for serving static files)
